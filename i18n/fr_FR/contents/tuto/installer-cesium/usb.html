Installation de Cesium Ğ1 sur clef USB et pour tout OS

Cesium peut être installé sur une clef USB pour l’avoir toujours à disposition. Attention cependant : utilisez toujours un ordinateur de confiance pour utiliser Cesium. Cette méthode peut également être utilisée pour installer Cesium sur des OS non supportés (Windows 32 bits, MacOS, etc.)

  • Téléchargez la version web.zip depuis cesium.app/telechargement
  • déplacez le fichier .zip vers la clef USB
  • extrayez l’archive en utilisant la méthode de votre choix :
    • clic droit > déziper ou clic droit > extraire,
    • logiciels 7zip, peazip
    • etc…
  • Pour lancer Cesium :
    • ouvrez le dossier Cesium
    • faites clic-droit sur index.html
    • ouvrir avec un navigateur (firefox, par exemple)
  • Si vous avez installé Cesium sur votre propre ordinateur et non sur une clef USB, vous pouvez maintenant l’ajouter dans les favoris en faisant un Ctrl+D. Quand vous taperez “Cesium” dans la barre d’URL, votre navigateur devrait le retrouver tout seul !
