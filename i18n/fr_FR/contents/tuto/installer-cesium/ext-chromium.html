Installer Cesium dans Chromium/Chrome

Ce tutoriel vous explique comment installer Cesium comme extension de navigateur pour Chromium/Chrome. Vous utiliserez Cesium dans le navigateur, mais pas sur un site web. Tout se passera dans votre ordinateur !

  1. Ouvrez Chromium/Chrome et rendez-vous sur la page de téléchargement
  2. Cliquez sur : Chromium/Chrome Add-on > Télécharger
  3. Cliquez sur "Add to Chrome", "Ajouter à Chrome".

  4. Acceptez l'installation avec "Add extension", "Ajouter l'extension".

  5. Et voilà ! Pour lancer Cesium, cliquez sur le logo Cesium en haut à droite.
