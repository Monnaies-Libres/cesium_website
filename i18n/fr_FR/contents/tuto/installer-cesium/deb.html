Installer Cesium Ğ1 sur GNU/Linux .deb (Debian, Ubuntu, etc.)

Cesium n’est pas encore inclus dans les dépôts officiels.

Pour l’installer sur votre système .deb, deux méthodes s'offrent à vous :

Méthode n°1 : En ligne de commande

Vous devez être connecté avec un compte ayant des droits “sudo”.

  1. Télécharger Cesium :
    wget lien/vers/cesium
  2. Installez-le (remplacez XXX par la bonne version)
    sudo dpkg -i cesium-desktop-vXXX.deb

S'il y a un souci à la première exécution, installez les éventuelle bibliothèque manquante :

sudo apt-get install libgconf-2-4

Méthode n°2 : Via l'interface graphique (logithèque Ubuntu)

  1. Si votre ordinateur tourne avec Ubuntu, la logithèque est normalement disponible.
  2. Téléchargez Cesium depuis la page téléchargement
  3. Ouvrez le dossier de téléchargement.
  4. Faites un double-clic sur le paquet cesium-desktop-vXXX.deb
  5. La logithèque va s'ouvrir. Cliquez sur "Installer"

Cesium derait finir de s'installer tout seul.

En cas de souci, installez le paquet libgconf-2-4 en utilisant votre gestionnaire de paquets favori
