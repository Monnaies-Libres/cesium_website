Installer Cesium dans Firefox

Ce tutoriel vous explique comment installer Cesium comme extension de navigateur pour Firefox. Vous utiliserez Cesium dans Firefox, mais pas sur un site web. Tout se passera dans votre ordinateur !
Si vous avez Firefox sur smartphone, ce tutoriel fonctionnera aussi.

  1. Si vous ne l'avez pas déjà, installez Firefox
  2. Ouvrez Firefox et rendez-vous sur la page de téléchargement
  3. Cliquez sur : Firefox Add-on > Télécharger
  4. Acceptez l'installation (deux boutons)


  5. Et voilà ! Pour lancer Cesium, cliquez sur le logo en haut à droite.
