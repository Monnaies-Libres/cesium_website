Installation de Cesium Ğ1 pour tout OS (éventuellement sur une Clef USB)

Cesium peut être installé sur une clef USB pour l’avoir toujours à disposition.

Attention cependant : utilisez toujours un ordinateur de confiance pour utiliser Cesium. Cette méthode peut également être utilisée pour installer Cesium sur des OS non supportés (Windows 32 bits, MacOS, etc.)

  1. Téléchargez la version web.zip depuis la page téléchargements
  2. Déplacez le fichier .zip vers l'endroit depuis lequel vous souhaitez l'utiliser (une clef USB par exemple)
  3. Extrayez l’archive en utilisant la méthode de votre choix :
    • clic droit > déziper ou clic droit > extraire,
    • logiciels 7zip, peazip
    • etc…
  4. Pour lancer Cesium :
    • Ouvrez le dossier Cesium
    • Faites clic-droit sur index.html
    • Ouvrir avec un navigateur (Firefox, par exemple)
  5. Si vous avez installé Cesium sur votre propre ordinateur et non sur une clef USB, vous pouvez maintenant l’ajouter dans les favoris en faisant un Ctrl+D. Quand vous taperez “Cesium” dans la barre d’URL, votre navigateur devrait le retrouver tout seul !
