Vous souhaitez installer Cesium sur votre machine ? C'est une très bonne résolution !

Des tutoriels détaillent la procédure pour plusieurs OS :

  • Gnu/Linux (.deb : Debian, Ubuntu, etc.)
  • Windows
  • Android
  • Firefox (tout OS)
  • iOS
  • Mac OS
  • sur clef USB (tout OS)

Vous pourrez ensuite créer votre compte.

Si vous rencontrez un problème ou avez une question, vous pouvez demander de l'aide :

  • sur le forum monnaie-libre.fr
  • sur le forum Duniter
